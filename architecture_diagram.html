Gradient vs Binary

GRADIENT (2 indicators)

Proportional scoring — stronger signal = more points. RS at 90th pctl gets more than 76th pctl.

BINARY (5 indicators)

All-or-nothing — full weight if condition met, zero if not. Based on 3-year conditional edge analysis.

subsector_breakout.py

Breadth-Based State Machine — Detect Sector Momentum

Aggregates individual ticker scores up to the subsector level, then runs a state machine to detect when an entire subsector is "breaking out."

METRICS COMPUTED
  • Breadth % — % of tickers scoring ≥ 7
  • Avg Score — subsector mean
  • Z-Score — vs 60-day rolling history
  • Acceleration — 2nd derivative of avg score
  • Hot Count — # tickers above threshold
TRANSITION TRIGGERS
  • Emerging: breadth ≥ 50% AND z > 1.0
  • Confirmed: hot for ≥ 3 consecutive days
  • Fading: breadth drops OR score declining
  • Back to Quiet: breadth < 30% OR cool ≥ 5 days
